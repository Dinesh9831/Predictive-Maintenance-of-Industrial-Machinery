## 📌 Overview  
A **Machine Learning project** that predicts industrial machine failures using sensor data.  
This system helps industries move from **reactive maintenance → proactive maintenance**, reducing downtime and operational costs.

---

## 🚀 Problem Statement  
Unexpected machine failures in industries lead to:  
- ⛔ Production downtime  
- 💸 Financial losses  
- ⚠️ Safety risks  

Despite having sensor data, identifying failure patterns remains a challenge.  

👉 This project solves it using predictive analytics.

---

## 💡 Solution  
We built a **Predictive Maintenance Model** that:  
- Analyzes machine sensor data  
- Detects hidden patterns  
- Predicts failure types before they occur

## 📊 Dataset  
- 📁 Source: Kaggle  
- 🔗 Predictive Maintenance Classification Dataset  
- 📌 Features include:  
  - Temperature  
  - Torque  
  - Rotational Speed  
  - Tool Wear  

---

## 🤖 Model Details  

### 🔹 Algorithm Used  
✅ Random Forest Classifier  

**Why?**  
- Handles complex data patterns  
- High accuracy  
- Robust against overfitting  

---

### 🔹 Training Strategy  
- 80/20 Train-Test Split  
- SMOTE applied to balance dataset  

---

### 🔹 Evaluation Metrics  
- ✔️ Accuracy Score  
- ✔️ F1-Score  
- ✔️ Confusion Matrix
